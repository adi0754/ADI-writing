# 快速开始指南

## 5分钟快速开始

### 1. 克隆并安装

```bash
git clone <repository-url>
cd h5-ai-writer
npm install
```

### 2. 运行开发服务器

```bash
npm run dev
```

打开 `http://localhost:5173`

### 3. 配置 API

首次使用时，选择您的 AI 提供商并输入 API 密钥：

**OpenAI:**
- 获取密钥: https://platform.openai.com/api-keys
- 需要: 有效的 OpenAI 账户

**Google Gemini:**
- 获取密钥: https://aistudio.google.com/app/apikey
- 需要: Google 账户

**DeepSeek:**
- 获取密钥: https://platform.deepseek.com/api
- 需要: DeepSeek 账户

### 4. 开始使用

1. 点击"新建文档"创建新的写作文档
2. 在文本框中输入或粘贴您的内容
3. 选择"扩写"或"润色"
4. 等待 AI 处理
5. 复制结果使用

## 常见问题

### Q: 我的 API 密钥安全吗？
A: 是的。您的 API 密钥仅保存在您的浏览器中，不会上传到任何服务器。

### Q: 我的文档会被保存吗？
A: 是的。所有文档都保存在您的浏览器的 localStorage 中。清除浏览器数据会删除它们。

### Q: 如何导出我的文档？
A: 在应用中复制文档内容，然后保存到您的文件中。

### Q: 支持哪些浏览器？
A: 所有现代浏览器 (Chrome, Firefox, Safari, Edge)

### Q: 需要付费吗？
A: 应用本身免费。但使用 AI API 可能会产生费用，取决于您的使用量。
